	R
	Python
	R, Python together

Deploy model delivery 
	File
	SQL table
	API call
	Web Form
	Native GUI application

Deploy in batch mode with 
	Source files
	Tables from db

Deploy model in event trigger mode using data from 
	API call reception
		REST API
	Steady stream
	Kafka queue
